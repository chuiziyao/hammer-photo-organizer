## ✨ 功能特性

- **一键整理**：选文件夹 → 点「开始整理」→ 自动按「年 / 月」归档
- **智能日期识别**：EXIF → 文件名 → 文件时间，逐级回退
- **自动去重**：按 MD5 比对，保留原片
- **整理后自动清场**：空掉的子文件夹自动移入回收站（可恢复）
- **按日期提取**：指定时间段导出，原文件一个都不动
- **会员无限量**：免费版前 5 次不限量、之后每次 100 张；会员扫码支付开通年卡 / 永久
