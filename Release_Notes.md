# 锤子照片整理工具 V2.0

一款**本地运行**的照片 / 视频整理工具：按拍摄日期把一团乱的照片视频，自动归档到「年 / 月」文件夹。支持智能去重、按日期提取、会员无限量整理。

> 软件纯本机运行，照片视频**不上传任何服务器**，隐私安全。

---

## ✨ 功能特性

- **一键整理**：选文件夹 → 点「开始整理」→ 自动按「年 / 月」归档
- **智能日期识别**：EXIF → 文件名 → 文件时间，逐级回退，识别不出也能手动补
- **自动去重**：按内容 MD5 比对，只删真正重复的字节，原片始终保留
- **整理后自动清场**：空掉的子文件夹自动移入回收站（可恢复），不留碎屑
- **按日期提取**：指定时间段导出到 `Extract_日期`，**原文件一个都不动**
- **会员无限量**：免费版前 5 次不限量、之后每次 100 张；会员扫码支付开通年卡 / 永久

## 🚀 使用步骤

1. 下载并运行软件（绿色版解压后双击 exe，安装版按向导装好）
2. 点「浏览」选择照片所在的最上层文件夹
3. 点「开始整理」，进度在日志区实时可见，几分钟搞定几千张
4. 整理结果在 `Organized_日期` 文件夹，按 `年/月` 排得整整齐齐
