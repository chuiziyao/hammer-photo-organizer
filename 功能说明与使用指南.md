## 一、核心功能

### 1. 照片 + 视频一起整理
自动识别 JPEG / PNG / TIFF / WEBP / HEIC / HEIF 及 RAW 原片（CR2 / CR3 / NEF / ARW / ORF / RW2 / DNG / RAW）等数十种照片格式，以及 MP4 / MOV / AVI / MKV / WMV / FLV / M4V / 3GP 与 `.mts / .m2ts / .ts / .vob / .rmvb` 等相机 / 摄像机格式（共 30+ 种媒体格式，其中视频 14 种）。

### 2. 按拍摄日期归档
读取照片 / 视频的拍摄时间（EXIF / 元数据 / 文件名），自动归入「年 / 月」文件夹；极少数冷门 RAW / HEIC 若无内嵌时间，则按文件名日期归类，无日期的单独归到 `Unknown` 文件夹。

### 3. 自动去重不误删
按内容 MD5 比对，仅删除真正重复的副本，原片始终保留。

### 4. 整理后自动清场
空文件夹自动移入回收站（可恢复），不留碎屑。

### 5. 本地处理不打扰
纯本机运行，照片视频绝不外传。

### 6. 按日期提取
点击「指定日期」，输入起止日期，可把某一时间段的照片 / 视频单独导出到 `Extract_日期` 文件夹，并按拍摄日期自动归类到「年 / 月」子文件夹，不改动原文件。

## 二、使用指南

### 第一步：选择文件夹
点击「浏览」或直接粘贴路径，选择要整理的相册根目录（建议选最上层目录，软件会递归扫描所有子文件夹）。

### 第二步：设置选项
- **备份文件（默认勾选）**：整理前自动备份，万一需要可找回原文件。
- **整理后清空空文件夹到回收站（默认勾选）**：整理完成后把空壳目录移入回收站。

### 第三步：开始整理
点击「开始整理」，软件会先扫描、再分析拍摄时间、最后按年 / 月归档到该目录下的 `Organized_日期` 文件夹。整理过程可在日志区查看进度。

### 第四步：按日期提取（可选）
点击「按日期提取」，输入起止日期，可把某一时间段的照片 / 视频单独导出到 `Extract_日期` 文件夹，并按拍摄日期自动归类到「年 / 月」子文件夹，不改动原文件。

### 第五步：编辑 EXIF（可选）
点击「编辑 EXIF」可查看并修改单张照片的拍摄时间、GPS 等信息（修改前建议备份）。

## 四、常见问题 FAQ

**Q：整理会删除我的原照片吗？**
A：不会。软件只把文件移动到 `Organized_` 文件夹并按日期归类；开启「备份」后还会额外留一份备份。去重只会删除内容完全相同的重复文件。

**Q：日期不准怎么办？**
A：优先读取 EXIF / 元数据；读不到时按文件名里的日期（如 `IMG_20260126`）或文件时间归类。若想校正，可用「编辑 EXIF」手动修改。

**Q：整理错了能还原吗？**
A：原文件未被删除，只是被移动并归类，可手动移回；开启备份时还能从备份目录恢复。

**Q：支持哪些语言？**
A：软件内置中文与英文，可在界面切换。
